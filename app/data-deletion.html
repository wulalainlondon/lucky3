Data Deletion Request

App: Lucky 3 Solitaire
Developer: Lucky3 Games
Effective Date: 2026-04-30

Overview

Lucky 3 Solitaire does not require an account. Most game progress, settings, achievements, and unlocks are stored locally on your device and can be deleted by uninstalling the app or clearing the app's local storage.

Leaderboard Data

If you choose to submit a Daily Challenge score, the app stores leaderboard data in Firebase. This may include an anonymous Firebase user ID, an optional display name, the challenge date, completion time, move count, combo information, result type, and update time.

How to Request Deletion

To request deletion of your Lucky 3 Solitaire leaderboard data, open a request at github.com/wulalainlondon/lucky3/issues.

Please include as much of the following information as you can, so we can find the correct leaderboard entry:

  • Your leaderboard display name, if you entered one.
  • The Daily Challenge date shown in the app, for example 2026-04-30.
  • Your approximate completion time or move count.
  • A screenshot of your leaderboard entry, if available.

What We Delete

After we verify the matching leaderboard entry, we will delete the leaderboard record associated with the request. This includes the anonymous Firebase user ID stored with that leaderboard entry, display name, score, move count, combo information, result type, and update timestamp.

Processing Time

We aim to process valid deletion requests within 30 days. If we need more information to identify the correct record, we may ask for clarification in the request thread.

Local Device Data

Local game data stays on your device and is not accessible to the developer. To delete local data, uninstall Lucky 3 Solitaire or clear the app's storage from your device settings.

Privacy Policy

For more information, read the Lucky 3 Solitaire Privacy Policy.
