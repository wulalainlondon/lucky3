📖 How to Play

Goal

Clear the board down to a single 3 card. Deal cards one at a time and eliminate sets of three.

Valid Sums

Select 3 cards in the same column whose values add up to:

9A-A-7 …
19A-9-9 …
299-10-10 …
♠A = 1  ·  2–10 = face value
Valid Positions

Only these 3 patterns are allowed (shown for a 5-card column):

A
B
C
💡Top cards = dealt first. Bottom card = most recent deal.
Recycle

When the deck runs out, tap RECYCLE to reshuffle discarded cards back into the deck. Each recycle costs one move.

Undo

Tap the ↩ button to reverse your last action (deal or clear). Use it wisely — each undo counts as a move.
