## 二、「Solitaire x Match 3」定調的核心問題

### 為什麼不能直接用

「Match 3」在市場上已是**強烈的視覺符號**：

> Candy Crush、Royal Match → 找三個相同圖案 → 視覺配對

你的核心是**數字相加**，不是視覺相同。用這個標籤會讓玩家帶著錯誤預期進場，前 30 秒就流失。

### 可以怎麼用

作為**跨界描述語**（不是主類型標籤），在描述文案中說：

> "It plays like solitaire, but clears like a match-3 — with numbers."

這樣是讓已知的兩個概念幫你定位新體驗，而不是誤導類型。

## 三、真正的差異化定位

你的遊戲有一個幾乎無競品的獨特組合：

```
紙牌接龍的耐心策略
+
數字湊算的邏輯快感
+
「幸運3」的文化共鳴
+
每日籤詩的情感黏著
```

這個組合在 App Store 現有遊戲中幾乎找不到直接競品。這才是應該強調的點。

## 五、關鍵字策略（ASO）

### 中文關鍵字（優先順序）
```
紙牌遊戲、接龍、消消樂、數字遊戲、益智、
每日挑戰、幸運、單機、無廣告、解謎
```

### 英文關鍵字
```
solitaire, card game, number puzzle, daily challenge,
patience, brain game, fortune, offline, no ads, match cards
```

**關鍵字策略原則：** 不要在標題塞「Match 3」，因為在 Match 3 搜尋結果裡會輸給 Candy Crush；但在描述文本中用「number-matching」可以被演算法捕捉到跨類型搜尋。

## 六、商店類別建議

| 平台 | 主類別 | 次類別 |
|---|---|---|
| App Store | 遊戲 › 紙牌 | 遊戲 › 益智 |
| Google Play | 紙牌 | 棋盤遊戲 |

**不要選「Match 3 / Casual」類別**，因為留存機制（每日挑戰、成就、籤詩）是中度玩家取向，不是輕度消消樂族群。在紙牌類別裡差異化反而更突出。

## 七、截圖規劃（最重要的行銷資產）

App Store 截圖決定 70% 的轉換率，建議如下排序：

| 張次 | 主視覺 | 標題文字 |
|---|---|---|
| 1 | 遊戲全貌（綠色桌面、四欄牌局） | 「湊出幸運 3，今天一局定運氣」 |
| 2 | 「咪牌」峰值瞬間（放大特寫） | 「第三張要落下了——是你的幸運嗎？」 |
| 3 | 勝利畫面（Lucky 3 + 籤詩） | 「今日幸運籤：紅心 3，你今天很有溫暖緣分」 |
| 4 | 每日挑戰排行榜 | 「每天同一副牌，全球同台競速」 |
| 5 | 成就頁面 | 「從新手到傳說，每個里程碑都留下來」 |
